# **App Name**: NurseCare Sri Lanka

## Core Features:

- Dashboard: Display next duty, monthly budget, monthly goal, and a daily motivational quote on a single screen.
- Duty Roster: Monthly calendar view to mark shifts, add hospital/ward details, and auto-count total duties and off days.
- Budget Manager: Set monthly income, track expenses across categories, and visualize spending with charts.
- Motivational Hub: Daily nurse-focused motivation and self-care reminders delivered through warm, empathetic messages.
- User Profile: Manage nurse details, shift preferences, goals, language, and notification settings.
- Smart Notifications: Reminders for upcoming duties and budget alerts to provide timely and gentle encouragement to nurses.
- AI Motivational Prompts: AI-powered tool to tailor daily motivational quotes based on user's profile and recent activity, enhancing the personalization and impact of the motivation hub.

## Style Guidelines:

- Primary color: Soft pastel blue (#A7D9ED), reflecting a calming and trustworthy feel.
- Background color: Desaturated light blue (#EBF4F8), maintaining the calming aesthetic while providing a neutral base.
- Accent color: Light green (#B3E2CD), complementing the blue tones and evoking a sense of health and growth.
- Body and headline font: 'PT Sans' for a modern yet warm and accessible feel. Note: currently only Google Fonts are supported.
- Use minimalistic and clear icons representing various modules (duty roster, budget, motivation).
- Clean and goal-focused dashboard design, with touch-friendly buttons, rounded cards, and intuitive navigation.
- Subtle animations for smooth transitions and loading states to enhance user experience.
